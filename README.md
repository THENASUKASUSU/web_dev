# Full Stack Portfolio Website

This is a full-stack portfolio website built with the MERN stack and Tailwind CSS.

## Features

-   React.js frontend with Tailwind CSS
-   Node.js and Express backend
-   MongoDB database for contact form submissions
-   Dark mode toggle
-   Splash screen with ASCII art
-   SHA256 anti-clone token in the console log
-   Audit log for contact form submissions
-   Theatrical black-and-red theme

## Local Setup

1.  **Clone the repository:**

    `git clone https://github.com/your-username/your-repo-name.git`

2.  **Install client dependencies:**

    `cd client && npm install`

3.  **Install server dependencies:**

    `cd ../server && npm install`

4.  **Create a `.env` file in the `server` directory and add your MongoDB connection string:**

    `MONGO_URI=your_mongodb_connection_string`

5.  **Run the client and server concurrently:**

    -   In one terminal, run the client: `cd client && npm start`
    -   In another terminal, run the server: `cd server && node server.js`

## Deployment

### Frontend (Vercel/Netlify)

1.  Push your code to a GitHub repository.
2.  Go to Vercel or Netlify and connect your GitHub account.
3.  Select your repository.
4.  Configure the build settings:
    -   **Build command:** `npm run build`
    -   **Publish directory:** `client/build`
    -   **Install command:** `npm install` (or `cd client && npm install`)
5.  Deploy the site.

### Backend (Render/Heroku)

1.  Push your code to a GitHub repository.
2.  Go to Render or Heroku and create a new web service.
3.  Connect your GitHub repository.
4.  Configure the build and start commands:
    -   **Build command:** `npm install`
    -   **Start command:** `node server.js`
5.  Add your `MONGO_URI` as an environment variable.
6.  Deploy the service.
7.  Update the `fetch` URL in your `client/src/components/Contact.js` file to point to your deployed backend URL.
